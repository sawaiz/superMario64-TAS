## Common reference hashes (verify your dump)

| Version | Format note | MD5 |
|---------|-------------|-----|
| Super Mario 64 (USA) | big-endian `.z64` | `20b854b239203baf6c961b850a4a51a2` |
| Super Mario 64 (Japan) | big-endian `.z64` | `85d61f5525af708c9f1e84dce6dc10e9` |

Byte-swapped `.n64` / `.v64` images have different MD5s but can be converted.

Many **1-key / any%** TASes use **Japan**. Many **IL / 120** archive files use **USA**. Match the movie.
